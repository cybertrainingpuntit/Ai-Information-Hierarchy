# Prompt 3 – Stel de Instructies‑file voor je eerste project op

Gebruik dit om het `Instructies.md`‑bestand van een project (bijv. een nieuwsbrief, YouTube‑serie, etc.) te vullen.

```
Ik wil het Instructies.md‑bestand voor mijn eerste project in mijn Informatie‑hiërarchie invullen.

Controleer eerst of je al context over dit project hebt uit eerdere gesprekken of uit mijn Memory.

- Als er voldoende context is, gebruik die dan om het Instructies.md‑bestand te schrijven en plaats het in [projectnaam]/Instructies/Instructies.md.
- Als ik een werkblad met deze informatie heb, plak ik dat hieronder en kun je dat gebruiken.
- Als er geen context is, interview me dan over dit project. Vraag me:
  * Waar gaat dit project over? (één zin)
  * Hoe ziet een geweldig resultaat eruit? Wees specifiek.
  * Wat moet de AI absoluut vermijden in dit project?
  * Waar bewaar ik referentiemateriaal, voorbeelden en notities voor dit project?

Zodra je alle benodigde informatie hebt – uit eerdere gesprekken, uit mijn Memory, uit een werkblad, of uit ons interview – schrijf je de inhoud van `Instructies.md`. Houd het kort en zonder corporate‑taal.

Na het opslaan van het bestand, vertel me:
1. Wat je hebt geschreven en waarom
2. Wat er eventueel mist of onduidelijk is en later nuttig zou kunnen zijn
```
